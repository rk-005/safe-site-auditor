About SafeSite Auditor

Welcome to SafeSite Auditor, an open-source ethical security tool designed to help you check the safety of websites and analyze suspicious emails.

Our mission is to provide easily understandable insights into potential web security vulnerabilities and common phishing indicators, promoting safer online practices.

Our Principles:

  • Ethical Auditing: This tool performs non-intrusive checks. It is strictly for educational and self-auditing purposes. Never use this tool to scan websites without explicit permission.
  • Privacy Focused: We do not store any URLs you scan or email content you provide. All analysis is performed on-the-fly and results are displayed client-side.
  • Transparency: The logic behind our checks is openly discussed to help you understand web security concepts.

How it Works:

Website Security Scan: Analyzes web headers, cookie flags, and looks for basic indicators of common vulnerabilities like XSS and SQL Injection without exploiting them. It identifies potential weak points that could be exploited by malicious actors.

Phishing Email Checker: Scans email content for suspicious keywords, malicious-looking links (without visiting them), and other common traits found in phishing attempts. It helps you identify whether an email is likely a scam.

SafeSite Auditor is a project built with Python (Flask) for the backend and standard web technologies (HTML, CSS, JavaScript) for the frontend. We welcome contributions and feedback!
